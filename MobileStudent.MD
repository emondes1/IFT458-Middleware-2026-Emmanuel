# Mobile App Notes (React Native)

## What is React Native
React Native is a framework used to build mobile applications using JavaScript and React. It allows developers to create cross-platform apps for both iOS and Android.

## Key Features
- Uses JavaScript and React
- Cross-platform development
- Reusable components
- Fast development with live reload

## How to Run a Mobile App
To run a React Native app, developers typically use tools like Expo or Android Studio. The app can be tested on an emulator or a real mobile device.

## Summary
React Native simplifies mobile development by allowing developers to write code once and deploy it across multiple platforms.
