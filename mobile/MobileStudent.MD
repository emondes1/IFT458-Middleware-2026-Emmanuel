# Mobile App Notes - Dan Warren

## What is React Native
- React Native lets you build mobile apps using JavaScript
- One codebase runs on both iOS and Android
- Uses native components instead of web views

## What is Expo
- Expo is a framework built on top of React Native
- Run `npx expo start` to launch the development server
- Press `w` to open the app in a web browser
- Scan the QR code with your phone to test on a real device

## What I Observed
- The mobile app is called "Book Exchange"
- App runs on http://localhost:8081
- Use browser DevTools device toolbar to simulate mobile screens
- Selected iPhone 14 profile to test mobile layout
- The app communicates with the backend running on port 4000
