# Mobile App Notes (React Native + Expo)

## What is React Native?
React Native is a framework that allows developers to build mobile applications using JavaScript and React.

## What is Expo?
Expo is a tool that simplifies React Native development by providing pre-configured environments and tools.

## How the Mobile App Works
The mobile app communicates with the backend server using API calls (HTTP requests).

## Running the App
To start the app:
npx expo start

Then press "w" to open in the browser.

## Backend Connection
The app connects to the backend running on:
http://localhost:4000

This allows the mobile app to fetch and send data.
